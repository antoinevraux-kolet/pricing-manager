# Pricing Manager

## Stack

- **Backend** — Node.js, TypeScript, Express, @google-cloud/bigquery
- **Frontend** — React, TypeScript, Vite

## Lancer l'app

### 1. Installer les dépendances

```bash
npm install
npm run install:all
```

### 2. Démarrer (backend + frontend simultanément)

```bash
npm run dev
```

Ou séparément :

```bash
npm run dev:backend   # http://localhost:3001
npm run dev:frontend  # http://localhost:3000
```
